# Barbershop Website

## Short Description

A full-featured barbershop website that allows customers to book appointments, shop for products, and enables staff to manage all business operations through an admin dashboard.

## Overview

This project provides barbershops with a complete online platform for managing their business. Customers can book appointments, browse services, and purchase grooming products, while staff manage everything through an admin dashboard with a built-in CMS. The goal was to create a self-sufficient solution that handles the entire customer journey without requiring ongoing developer involvement.

## Features

- Book appointments with a multi-step wizard that lets customers choose their service, preferred barber, and available time slot
- Browse and purchase grooming products through the integrated e-commerce store with shopping cart and checkout
- Submit contact inquiries that are validated and stored for staff to review
- Switch between dark and light themes based on preference or system settings
- Manage all website content through a comprehensive admin dashboard
- Create, edit, and publish pages with a built-in CMS editor
- Handle product inventory with stock tracking and order management
- View and organize customer appointments and update their status
- Upload and organize images through a media library
- Manage staff profiles, services, and pricing

## Tech Stack

- **Languages:** HTML5, CSS3, JavaScript (ES6+ Modules)
- **Backend:** Firebase (Firestore, Authentication, Storage, Cloud Functions)
- **Libraries:** Flatpickr (date/time picker), ScrollReveal (animations)
- **Fonts:** Google Fonts (Playfair Display, Inter, Roboto)
- **Runtime:** Node.js 20 (Cloud Functions)
- **Hosting:** Firebase Hosting, Vercel
- **Tools:** Firebase CLI, ESLint
