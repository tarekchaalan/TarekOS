# Kuwait Driving Test

## Short Description

A native iOS app that helps users prepare for the Kuwait driving theory exam by practicing real Ministry of Interior (MOI) questions.

## Overview

Kuwait Driving Test was created to address the lack of accessible, high-quality practice tools for the Kuwait driving theory exam. The app provides an authentic test experience using official MOI questions, supporting both English and Arabic languages. Users can practice through multiple quiz modes, study with flashcards, bookmark difficult questions, and track their progress over timeâ€”all within a polished, dark-themed interface designed for focused learning.

## Features

- Practice through multiple quiz modes including Standard, True/False Only, Images Only, Critical Only, and Difficult
- Study with flashcard-style navigation and answer reveal
- Bookmark questions for focused review sessions
- Switch between English and Arabic with full bilingual support
- Identify traffic signs through image-based recognition questions
- Experience critical question rules with instant-fail scoring that mirrors the real exam
- Track result history with an analytics dashboard showing attempts, pass rate, and average score
- Celebrate perfect scores with confetti animations
- Browse a dark theme with modern glassy UI design
- Navigate with haptic feedback and VoiceOver accessibility support

## Tech Stack

- **Language:** Swift 5.9+
- **Framework:** SwiftUI
- **Platform:** iOS 18+ / iOS 26
- **IDE:** Xcode 15+
- **Data Format:** JSON (question bank)
- **Architecture:** MVVM (Model-View-ViewModel)
